# Chuckle
An assistant to handle questions related to data from child monitoring app Huckleberry. Queries are also supported with information from the NHS about [baby care](https://www.nhs.uk/baby/).

## Capabilities

### Guidance and Support

1. Answer questions related to child using data ingested from Huckleberry [Memory Jogger]
2. Provide supoprt on general queries related to childcare based on NHS information


#### Supported Queries
For example:

1. Why is my baby waking up again?
2. How do I know my baby's ready 


More information about Huckleberry's official AI is [here](https://huckleberry.zendesk.com/hc/en-us/articles/44561361627667-What-is-Berry)
